# 🐍 Hola Snake! (像素西语贪吃蛇)

> 一款运行在“复古掌机”上的贪吃蛇游戏，让你在吃水果的同时学习西班牙语单词。

## 📖 简介 (Introduction)

**Hola Snake!** 是一个基于 React 构建的复古像素风格网页游戏。它完美复刻了经典 GameBoy 掌机的外观与手感。玩家控制一条贪吃蛇在屏幕上移动，目标是吃掉随机出现的水果。

但这不仅仅是一个游戏！每当你吃掉一个水果，屏幕上就会显示其对应的 **西班牙语单词**。游戏还包含一个“图鉴系统”，记录你遇到的每一种水果。

## ✨ 特性 (Features)

*   **🕹️ 沉浸式复古体验**：精美的 CSS 像素风掌机外壳，配合真实的按键音效和 8-bit 背景音乐。
*   **🇪🇸 寓教于乐**：内置西语词库，吃水果学单词（Manzana, Plátano, Naranja...）。
*   **📚 图鉴系统 (Gallery)**：查看已解锁的水果像素画和单词卡片。
*   **⚡ 动态交互**：
    *   **B 键 (红色)**：按住加速移动。
    *   **A 键 (绿色)**：开始游戏 / 游戏结束后一键重开。
    *   **Start / Select**：控制游戏流程。
*   **📱 响应式控制**：支持键盘操作和屏幕虚拟按键点击。

## 🎮 操作指南 (Controls)

| 动作 | 键盘按键 | 掌机按钮 |
| --- | --- | --- |
| **移动** | 方向键 / W,A,S,D | 十字键 |
| **加速** | J 键 / 空格键 | **B 键 (红色)** |
| **确认/重开** | K 键 / Enter | **A 键 (绿色)** |
| **开始/暂停** | Enter | **START** |

## 🛠️ 技术栈 (Tech Stack)

*   **核心框架**: [React 18](https://reactjs.org/)
*   **构建工具**: [Vite](https://vitejs.dev/)
*   **样式**: CSS3 (Flexbox, Grid, CSS Animations)
*   **字体**: "Press Start 2P" (Google Fonts)

## 🚀 如何运行 (How to Run)

1.  **安装依赖**:
    ```bash
    npm install
    ```

2.  **启动开发服务器**:
    ```bash
    npm run dev
    ```

3.  打开浏览器访问显示的本地地址（通常是 `http://localhost:5173`）。
